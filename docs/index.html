Getting Started

Embed the script in your pages! It's that easy.

<script type="text/javascript" src="path_to_ntfy.min.js"></script>
<script>ntfy.init(),ntfy.ntfy(type,position,title,message);</script>

The Components & Customizations

Notify provides pre built icons and notifications accordingly.

Types

success

Display a success message if anything's done right.

error

Display a error message if something went wrong.

warning

Display a warning when the user does something out of the box.

info

Display a help/info about a particular action.

notification

Any important notification or any alerts push them right now.

date

Display the date or anything just related to date/time.

pending

Display a pending status when something's not yet confirmed.

fav

Bookmary this library and add it as a fav.

printing

Sit tight! We are printing your application or your tickets.

filecopy

The files have been copied from someone else's computer.

Position

top-left

Display the toast at the top left corner.

top-right

Display the toast at the top right corner.

bottom-left

Display the toast at the bottom left corner.

bottom-right

Display the toast at the bottom right corner.

top-center

Display the toast at the top center.

bottom-center

Display the toast at the bottom center.
